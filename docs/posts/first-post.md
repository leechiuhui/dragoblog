# 我的第一篇文章

歡迎來到我的 VitePress 部落格！

## 為什麼選擇 VitePress？

VitePress 是一個靜態網站生成器，具有以下優點：

1. **快速** - 基於 Vite 的極速開發體驗
2. **簡單** - 使用 Markdown 編寫內容
3. **靈活** - 可自訂主題和元件
4. **SEO 友好** - 靜態生成的 HTML

## 開始寫作

使用 VitePress 寫作非常簡單：

```markdown
# 標題
內容...
```

就是這麼簡單！
